🛸 AltDünya Blog Yazısı Hazırlayıcı

Sadece yazıyı hazırlar. Siteye otomatik eklemez. Çıkan JSON bloğunu data/blog.json içine yapıştırırsın.

Boş satırlar otomatik paragraf yapılır. HTML bilmen gerekmez.

Not: Birden fazla yazıyı otomatik birleştirme yok. Bu özellikle sade tutuldu.

JSON çıktısı burada görünecek.

Kullanım: JSON Oluştur → Kopyala → data/blog.json dosyasında köşeli parantezlerin içine yeni obje olarak ekle.
